## steps to install app

**Create python virtual environment**

```python<version> -m venv <env name>```
**activate env**
```source mcqui/bin/activate ```

***Install Dependecies***

```pip install -r  requirements.txt ```

note: check selected interpreter in your IDE 

***Run app***

```python app.py```
